Base R Graphics Fundamentals

Primary & Secondary Plot Functions • Graphical Parameters • Layouts

📊 1. Primary Plot Functions (High-Level)

These functions create complete plots from scratch.

Basic Plot Types

  • plot(x, y, type="p") — Scatter/line chart
  • hist(x) — Histogram
  • barplot(height) — Bar chart
  • boxplot(x ~ group) — Box plot
  • pie(x) — Pie chart
  • dotchart(x) — Cleveland dot plot
  • stripchart(x) — 1D scatter plot

Plot Type Parameter

  • type="p" — Points (default)
  • type="l" — Lines
  • type="b" — Both points & lines
  • type="o" — Overplotted
  • type="h" — Histogram-like vertical lines
  • type="s" — Stair steps
  • type="n" — No plotting (setup)

Formula Interface

  • plot(y ~ x, data=df) — Formula scatter
  • boxplot(value ~ group, data=df) — Grouped
  • coplot(y ~ x | factor, data=df) — Conditioning plots

Special Purpose Plots

  • pairs(df) — Scatterplot matrix
  • matplot(matrix) — Plot matrix columns
  • curve(sin(x), 0, 2*pi) — Plot a function
  • contour(volcano) — Contour plot
  • image(matrix) — Heat map
  • persp(z=volcano) — 3D perspective plot

✏️ 2. Secondary Plot Functions (Low-Level)

These add elements to existing plots.

Adding Data Elements

  • points(x, y) — Add points
  • lines(x, y) — Add lines
  • abline(a, b) — Line (intercept, slope)
  • abline(h=y) — Horizontal line
  • abline(v=x) — Vertical line
  • segments(x0,y0,x1,y1) — Segments

Shapes & Annotations

  • arrows(x0,y0,x1,y1) — Arrows
  • rect(xleft,ybottom,xright,ytop) — Rectangles
  • polygon(x, y) — Polygon
  • text(x, y, labels) — Add text at coords
  • mtext("text", side) — Margin text
  • Sides: 1=bottom, 2=left, 3=top, 4=right

Reference & Decoration

  • title(main, sub, xlab, ylab) — Titles
  • legend(x, y, legend, ...) — Legend
  • grid() — Add grid
  • axis(side, at, labels) — Custom axis
  • rug(x) — Rug plot on axes
  • box() — Draw box around plot
  • clip(x1,x2,y1,y2) — Set clipping

⚙️ 3. Graphical Parameters (par())

Set globally with par() or within individual function calls.

Color Parameters

  • par(col="red") — Default color
  • par(col.axis="blue") — Axis annotation
  • par(col.lab="darkgreen") — Axis labels
  • par(col.main="purple") — Main title
  • par(col.sub="gray") — Subtitle
  • par(bg="lightyellow") — Background
  • par(fg="black") — Foreground

Symbol & Line Types

  • par(pch=19) — Point character (0-25)
  • par(lty=2) — Line type
  • lty: 1=solid, 2=dashed, 3=dotted
  • par(lwd=2) — Line width (default=1)
  • par(cex=1.5) — Symbol/text expansion

Text Parameters

  • par(cex.main=1.2) — Title expansion
  • par(cex.lab=1) — Axis labels expansion
  • par(cex.axis=0.8) — Tick labels
  • par(font=2) — Font style
  • font: 1=normal, 2=bold, 3=italic, 4=bold italic
  • par(family="serif") — Font family

Axis & Margin Parameters

  • par(las=2) — Label style (0-3)
  • par(mgp=c(3,1,0)) — Axis margin lines
  • par(tck=-0.01) — Tick mark length
  • par(xlog=TRUE) — Log x-axis
  • par(lab=c(5,5,7)) — # of axis intervals

Margin & Size Parameters

  • par(mar=c(5,4,4,2)) — Margins (b,l,t,r)
  • par(mai=c(1,0.8,0.8,0.4)) — In inches
  • par(oma=c(0,0,0,0)) — Outer margins
  • par(pin=c(4,3)) — Plot dimensions (in)
  • par(pty="s") — Square plot type
  • par(pty="m") — Maximal plot type

📐 4. Layouts & Multiple Plots

Arranging multiple plots on one device.

mfrow & mfcol

  • par(mfrow=c(2,2)) — 2×2, filled by rows
  • par(mfcol=c(2,2)) — 2×2, filled by columns
  • par(mfrow=c(1,1)) — Reset to single plot
  • Plots are filled sequentially

layout() — Complex Grids

  • layout(matrix(c(1,1,2,3), 2, 2))
  • Matrix defines plot positions
  • widths = c(2,1) — Column ratios
  • heights = c(1,2) — Row ratios
  • layout.show(n) — Preview grid
  • Reset: par(mfrow=c(1,1))

split.screen()

  • split.screen(c(2,1)) — Split 2 rows
  • screen(1) — Select screen
  • screen(2) — Select second screen
  • close.screen(all=TRUE) — Clean up
  • Allows non-grid arrangements

🖱️ 5. Interactive Elements & Key Points

Interactive functions and essential best practices.

Interactive Functions

  • locator(n=3) — Click for coordinates
  • identify(x, y, labels) — Label points

Key Points to Remember

  • High-level functions create new plots
  • Low-level functions add to current plot
  • Parameters: global with par() or in-call
  • Use mfrow/layout() for multiple plots
  • Always reset parameters after customization
  • Colors: names, hex, rgb(), palettes

📝 Complete Example: Combining Concepts

# Setup layout and parameters
par(mfrow = c(2, 2), mar = c(4, 4, 3, 1))

# Plot 1: Scatter with regression
x <- 1:50; y <- 2*x + rnorm(50, 0, 10)
plot(x, y, main = "Scatter", pch = 19,
     col = rgb(0.2, 0.4, 0.6, 0.7))
abline(lm(y ~ x), col = "red", lwd = 2)
legend("topleft", "Regression", col="red", lty=1)

# Plot 2: Histogram with density
hist(y, prob = TRUE, col = "lightblue")
lines(density(y), col = "darkred", lwd = 2)

# Plot 3: Box plot
boxplot(y, col = "lightgreen", horizontal = TRUE)

# Plot 4: Reset
plot(0:10, 0:10, type = "n", main = "Custom")
rect(2, 2, 8, 8, border = "blue", lwd = 2)

# Reset to defaults
par(mfrow = c(1, 1), mar = c(5, 4, 4, 2))
