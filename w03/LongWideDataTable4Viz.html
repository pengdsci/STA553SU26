Long vs Wide Format · When to Use What

📋

Wide Format

“Human & Summary Table”

Example: One row per store, columns for Jan/Feb/Mar revenue.

🧬

Long (Tidy) Format

“Machine & Code Ready”

Example: Each row is one observation (Store-Month-Revenue).

✨

Golden Rule

Start with long format for BI tools & code.

Exception: Excel & correlation matrices need wide.

🔀

Quick Decision Flow

QuestionFormat
Correlation matrix or Excel chart?Wide
Printed report table?Wide
Using code / Tableau / BI?Long
Multi-series line/bar chart?Long

✅ Checkboxes above are interactive.

📌 At a Glance

Use caseBest format

🧠 Remember

“If your data is wide, pivot longer before visualizing.”

Most tools expect tidy data. Use pivot_longer() in R or melt() in pandas.
