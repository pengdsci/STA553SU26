🧹 1. Data Cleaning & Preparation

Core tasks for visualization-ready data: removing unnecessary variables, duplicate rows, outliers, missing values, standardizing values, and typo correction.

🔗 2. Merging Data Sets

base R dplyr Combine relational data using keys. Inner, Outer (Full), Left, Right joins with merge() and dplyr mutating joins.

⚙️ base::merge()

inner_join <- merge(employee, city, by.x='EmpID', by.y='ID', all=FALSE)
left_join  <- merge(x, y, all.x=TRUE)
right_join <- merge(x, y, all.y=TRUE)

✨ dplyr joins

inner_join(x, y, by="key")
left_join() , right_join() , full_join()

Pipe operator %>% chains operations: employee %>% inner_join(city, by=c("EmpID"="ID"))

✂️ 3. Subsetting Data

Select rows (conditions) and columns. Accessors: [, [[, $ . Base R which() and dplyr::filter() + select().

Example – subset setosa species: setosa.id <- which(iris$Species == "setosa"); setosa.flower <- iris[setosa.id, ]
With dplyr: filter(iris, Species == "setosa")
Multiple conditions: filter(iris, Species == "setosa" & Sepal.Length > 5)
Using .data pronoun: dynamic column selection

📦 4. Data Wrangling with dplyr

Key verbs: select(), filter(), mutate(), group_by(), summarize(), arrange(), count(). Powerful for analytical workflows.

# Grouped summary example:
iris %>%
  filter(Petal.Length < 5) %>%
  group_by(Species) %>%
  summarise(avg_sepal_width = mean(Sepal.Width),
            cor_sepal = cor(Sepal.Length, Sepal.Width)) %>%
  arrange(desc(cor_sepal))

🔄 5. Reshaping Data (tidyr)

gather() → wide to long (stack columns). spread() → long to wide. Essential for visualization and modeling.

# Example: life expectancy from wide to long
life_long <- life_expectancy %>%
  gather(key = "Year", value = "lifeExp", -country, na.rm = TRUE)

# spread back
mini_wide <- mini_long %>% spread(key = "flower.att", value = "measurement")
