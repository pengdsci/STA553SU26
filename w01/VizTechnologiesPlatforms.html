STA 553 Data Visualization

Modern Technologies & Platforms

A comprehensive overview of tools for visual storytelling, dashboards & web-based presentation

🐙
GitHub & GitHub Pages
Hosting & Collaboration
Version Control Static Hosting Portfolio
  • Version control for visualization code and data
  • Free static web hosting via GitHub Pages
  • Showcases interactive portfolios and project demos
💻
RStudio & RPubs
Development & Publishing
IDE R Markdown One-Click Publish
  • Primary IDE for R with integrated preview tools
  • Visual editor for R Markdown and Shiny apps
  • RPubs: free one-click publishing of HTML reports
📊
Shiny & Flexdashboard
Interactive Web Apps & Dashboards
Reactive Apps Dashboards R-Based
  • Build interactive web apps directly from R
  • Flexdashboard: row/column dashboard layouts
  • Embed Shiny components and HTML widgets easily
🎨
ggplot2 & plotly
Core Visualization Libraries
Grammar of Graphics Interactive Layered
  • ggplot2: declarative, layer-by-layer plot building
  • plotly: converts ggplot2 charts to interactive web graphs
  • Built-in hover, zoom, pan without JavaScript
📈
Tableau
Business Intelligence & Analytics
Drag & Drop Enterprise No-Code
  • Leading drag-and-drop analytics platform
  • Connects to diverse data sources seamlessly
  • Prioritizes visual best practices for dashboards
🌐
Web Slides & HTML/CSS
Presentation & Styling Libraries
xaringan Quarto CSS Customization
  • xaringan & revealjs: HTML5 slides from R Markdown
  • Quarto: next-gen multi-language publishing system
  • HTML/CSS for fine-tuned styling beyond defaults
