✅ Core hypotheses & action tracker (click to mark as reviewed)

📉 Expected portfolio loss under 200bps interest rate shock

Scenario: 200 bps increase in base rates → higher monthly payments for variable-rate & refinancing pressure. Estimated default rate uplift: +3.2pp, taking baseline from 14.2% → 17.4% expected loss rate in near-prime segment.

Stress test also incorporates DTI sensitivity: borrowers with DTI >40% would see 5pp higher default migration. Recommendation: dynamic underwriting and macro overlay.

⚠️ Caveats: Survivor bias from approved applicants only; macroeconomic variables not in model; static bureau data limited. Consider reject inference.
📋 Strategic actions & monitoring roadmap

  • Monitoring: Vintage performance dashboard with auto-alert: new vintage 6-month default rate > trailing 12-month avg +100bps.
